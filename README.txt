to run the project follow these steps:

open visual studio code

open the project folder so you can see all files and folders inside of it

head over to the terminal tab below , and type 

python -m venv env

then 

.\env\Scripts\activate

then 

pip install -r requirements.txt

it should install all required libraries

after that type " python app.py " in the terminal 

check in the output and it should give you a link with a port , something like this ''http://127.0.0.1:5000/''

click on it and you should be able to try the demo
